📅 2025-04-11 | 新增离线浏览功能 🚀

全站支持离线访问模式现已上线！ 使用兼容浏览器即可实现无网络环境下的内容访问。

🛠 功能使用指南

第一步：使用兼容浏览器访问

  • ✅ 推荐浏览器：
    • Chrome（Android/PC）
    • Edge（Android/PC）
    • Safari（iPhone）
  • ❌ 暂不兼容：
    • UC浏览器
    • QQ浏览器

第二步：离线访问流程

  1. 首次访问：系统自动缓存页面资源（含图文内容）
  2. 离线模式：
    • 直接输入网址（www.zhonmiaozhimen.cn） ；或通过收藏的书签访问即可
    • 已添加桌面图标的用户可直接点击图标启动
  3. 内容更新：
    • 联网状态下后台自动同步最新内容
    • 长期未访问用户建议：联网打开等待5分钟 → 关闭 → 重新访问

📱 进阶操作 桌面应用快捷方式创建教程（非必要）

移动端操作

Android 设备：

  1. 使用 Chrome/Edge 打开网站
  2. 点击浏览器菜单（⋮）
  3. 选择「添加到主屏幕」/「添加到手机」
  4. 按提示完成设置（需授予桌面图标权限）

iOS 设备：

  1. Safari 打开网站
  2. 点击分享按钮（↑）
  3. 选择「添加到主屏幕」
  4. 自定义名称后确认

电脑端操作

  1. 使用 Chrome/Edge 访问网站
  2. 点击地址栏「安装」图标
  3. 或通过菜单（⋮）选择「安装」

⚠️ 重要提示

  • 数据清除：卸载应用/清理浏览器缓存将删除离线内容
  • 优化建议：建议在 WiFi 环境使用几分钟来完成首次完整缓存
